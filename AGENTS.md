# AGENTS.md

## 目的

このリポジトリは、FX 向けのローカル完結型クオンツ取引研究/自動売買デスクトップアプリを維持・拡張するためのものです。

## アーキテクチャ要点

- UI は PySide6 のネイティブデスクトップを維持する
- エンドユーザー向け文言は必ず日本語
- 戦略/特徴量/シミュレーションは UI から分離する
- `LocalSimBroker` と将来の GMO private broker は同一インターフェースで扱えるようにする
- 実売買ブローカーは存在しても既定では無効のままにする

## コーディング規約

- Python 3.11
- `src/` レイアウト
- 型ヒント必須
- 純粋ロジックと I/O を分離
- 時刻は timezone-aware を徹底
- ユーザー向け例外は日本語で明確に
- 巨大ファイル化を避け、責務単位で分割する

## モジュール境界

- `features/` は将来データソース変更があっても再利用できる純粋ロジックを保つ
- `strategies/` はシグナル説明文まで返す
- `simulation/` はバックテストの主系であり、簡易検証用ライブラリに依存しない
- `desktop/` は重い処理を直接実装せず `application.py` 経由で呼ぶ

## テスト期待値

- 設定ロード
- データキャッシュ
- 指標/パターン/ノールックアヘッド
- 戦略判定
- ポートフォリオ会計
- リスク制約
- レポート生成
- CLI デモ/バックテストのスモーク

## 安全ルール

- ライブ取引を既定で有効化しない
- 将来の実売買ブローカー安全ゲートを緩めない
- UI 上の注意文を削除しない
- デモ/実時間シミュレーション/将来の実売買の挙動差を README とヘルプで明示する

## 将来タスクへの指針

- 既存のモジュール境界を壊さず拡張する
- 日本語 UI 文言を優先し、内部識別子だけ英語を維持する
- まずローカルシミュレーションと再現性を守り、その後にブローカー接続を広げる
